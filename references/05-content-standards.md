# 05-content-standards

读取时机：当任务涉及内容核查、文案提炼、叙事骨架、内容差距、讲者备注、演讲稿、报告转演示或把资料重构成页面时读取。需要按项目类型选择叙事结构时，同时读取`14-narrative-patterns.md`。

## 内容核查

内容副主编必须检查原始资料是否足以支撑交付目标。关键缺失可以从已有资料提炼、用明确假设补写，或列入待补充问题。

## 叙事骨架

| 项目类型 | 推荐叙事骨架 |
| :--- | :--- |
| 授课培训 | 为什么学 → 学什么 → 怎么做 → 案例 → 练习 → 总结 |
| 客户汇报 | 背景 → 问题 → 洞察 → 方案 → 价值 → 路径 → 下一步 |
| 商业路演 | 机会 → 痛点 → 产品 → 市场 → 增长 → 商业模式 → 壁垒 → 团队 → 融资用途 |
| 内部汇报 | 目标 → 进展 → 结果 → 问题 → 原因 → 风险 → 下阶段计划 |
| 产品发布 | 场景痛点 → 新能力 → 核心亮点 → 使用场景 → 证明 → 行动号召 |
| 研究报告 | 研究问题 → 方法 → 数据 → 发现 → 解释 → 结论 → 建议 |

| 项目类型 | 必查内容 |
| :--- | :--- |
| 客户汇报 | 客户背景、问题、洞察、方案、证据、建议、下一步 |
| 授课培训 | 学习目标、知识结构、案例、练习、总结、讲者提示 |
| 销售提案 | 痛点、诊断、方案、价值、案例、实施、商务条款、下一步 |
| 商业路演 | 市场、痛点、产品、增长、商业模式、壁垒、团队、融资用途 |
| 内部汇报 | 目标、进展、结果、偏差、原因、风险、资源需求、计划 |
| 内部复盘 | 事实、原因、影响、责任、动作、时间表 |
| 产品发布 | 用户、场景痛点、能力、卖点、Demo路径、证明、行动号召 |
| 研究报告 | 研究问题、方法、数据来源、发现、解释、结论、建议 |

## 页数充分性判断

语言内容副主编必须主动判断页数是否足以承载叙事，而不是只在用户已给页数、旧版页数、样本页数或模板页数内压缩内容。页数是内容结构、文案密度、审美呼吸、UI承载和动效节奏共同决定的结果。

必须拆页或增页的信号：

1. 一页包含两个以上核心观点，标题无法只承担一个判断。
2. 案例事实、判断依据和复盘结论混在同一页，观众无法跟随推理。
3. 流程阶段、面试链路、录用后动作等语义阶段被合并，只是为了省页。
4. 练习页没有足够空间放题干、规则、填写区、反馈区和讲师提示。
5. 表格或清单因页数不足被压成小字、窄栏、随机换行或卡片堆叠。
6. 分步讲述需要 suspense，但页面初始态已经露出答案或后续步骤。
7. 审美留白、视觉锚点或投屏可读性需要独立页面承载。

必须合页或删页的信号：

1. 连续页面只有同类弱观点，拆开后没有新的受众反应。
2. 页面只是过渡标签，没有讲述动作、证据或视觉任务。
3. 同一案例被重复铺陈，合并后不影响推理链。
4. 模板或旧版留下的占位页没有当前内容支撑。

## 页面文案规则

1. 每页只表达一个核心观点。
2. 标题负责判断，不只做主题标签。
3. 正文负责证据和解释，避免把原文整段搬入页面。
4. 讲者备注可承载解释、转场和问答，不要塞进页面正文。
5. 关键数字、案例、结论必须保留来源或口径。
6. 培训只是内容类型之一，不得把客户汇报、路演、研究报告等误改成课程结构。

## 内容创造到UI/动效的转译责任

语言内容副主编不是只负责“把字写好”。正式视觉生产前，内容部门必须先把内容关系转译为UI和动效设计输入，再交给版式/UI、品牌审美和动效设计部门。没有内容转译，UI不得凭审美偏好直接排版。

必须判断每页属于哪种内容关系：

| 内容关系 | UI结构建议 | 动效/静止建议 | 内容部门必须给出的设计输入 |
| :--- | :--- | :--- | :--- |
| 判断 + 证据 | 大判断 + 证据旁注/截图/数据 | 证据先出或判断先出，按讲述意图决定 | 哪句话是判断，哪项证据最能支撑 |
| 对比/冲突 | 左右对照、前后对照、风险/收益分区 | 两侧分步揭示或高亮差异 | 对比维度、胜负关系、不能混淆的边界 |
| 流程/路径 | 时间线、阶段路径、泳道、闭环 | 按步骤出现，保留当前位置 | 阶段名、触发条件、下一步动作 |
| 案例复盘 | 原始事实区 + 解释区 + 结论区 | 事实先出现，结论后揭示 | 事实、判断、可验证证据、追问 |
| 工具/工作台 | 输入区、操作区、输出区、检查区 | 按使用动作分步 | 用户要填什么、系统给什么、何时判断 |
| 练习/互动 | 题干、限制、填写区、反馈区 | 先题干后规则，答案延迟 | 学员动作、时间、判断标准 |
| 收束/行动 | 大结论 + 行动清单 + 时间/责任 | 结论稳定，行动逐条出现 | 观众离场后要做的唯一动作 |
