# 18-history-derived-governance

读取时机：当用户要求修订本Skill、指出“同一个问题反复出现”、质疑subagent/副主编/模板/参考库/流行调研/审美/动效/QA没有真实发挥作用，或要求“基于历史对话全面修订”时必须先读。本文件来自用户在历史对话中的反复退回意见，是高于普通生产偏好的硬门禁补丁。

## Subagent持续授权规则

一旦用户在同一项目中质疑或要求过“subagent/子agent/多Agent/副主编真实参与/团队分工/不要主Agent自审”，后续同一项目的生产、修复、审稿、QA、Style Frame和Skill治理默认视为已授权创建subagent。不得在后续轮次重新以“未明确要求subagent”为理由降级。

完整视觉生产最小真实Agent编制：

| 能力域 | 必须输出 | 否决权 |
| :--- | :--- | :--- |
| 需求/交付 | 项目类型、受众、交付范围、禁止事项 | 目标不清或格式错误 |
| 语言内容 | 内容差距、叙事蓝图、每页核心判断、训练动作 | 内容被模板扭曲 |
| 品牌审美/趋势 | 参考库检索、趋势取舍、采用/弃用理由 | 风格套用、参考不适配 |
| 版式/UI | 页面族谱、组件语法、模板改造差异 | 模板先行或组件不服务内容 |
| 动效设计 | 动效脚本、静止页、分步揭示、导出稳定态 | 统一动画或遮挡阅读 |
| 交互/工程 | 实现约束、模式、截图/导出策略 | 控件破坏画面或无法稳定导出 |
| QA | 截图、人工审美复核、冻结/阻断 | 烟测冒充QA或证据不足 |

## 深度定制流程

不得从模板、风格或设计引擎开始。每个视觉生产任务必须按此顺序：

1. 当前资料盘点：内容、受众、场景、品牌、证据、训练/讲述动作。
2. 内容深挖：先填写《内容深挖表》，找出不可被模板压扁的重点。
3. 页数充分性判断：填写《页数充分性判断表》，由内容、文案、审美、UI、动效、交互/交付和QA判断是否需要增页、拆页、合页、删页、附录或讲者备注承载。
4. 页面族谱：填写《页面族谱表》，封面、讲授页、模型页、案例剧场、工作台、AI复核页、收束页等由内容关系生成。
5. 参考库/趋势检索：列候选经验、采用经验、弃用经验、弃用理由。
6. 内容到UI/动效转译：每页说明内容关系、UI结构、视觉锚点、动效/静止理由。
7. 部门联合审查：内容、UI、审美、动效、参考库、流行调研、交互、QA均有通过/退回/否决结论。
8. 模板候选：仅作为实现候选，不能作为设计方向。
9. 模板适配性审查：证明适合当前内容；不适合则弃用。
10. 模板改造差异表：即使适合，也必须逐页改造。
11. 禁止硬套自检：确认没有先选风格、模板填空、主Agent自签等情况。
12. Page Spec：每页的核心观点、证据、结论、视觉锚点、版式策略、禁用元素。
13. Style Frame：至少2个方向或按任务等级要求，截图必须晚于协同输入。
14. 生产、复核、QA、用户验收。

## 失败后停产规则

用户出现以下任一退回信号时，当前视觉产物立即判定为用户验收未通过，并进入`FAILED_VISUAL_FREEZE`：

- 质疑是否真实调用subagent、多轮复核、参考库或Skill。
- 明确指出丑、难看、不舒服、像模板、排版混乱、色彩失败。
- 指出多版本反复失败、浪费token、流程不可信。

停产后禁止动作：

- 继续生成下一版HTML/PPT/PDF或截图包。
- 创建Vx目录、复制旧版作为新版本、重命名失败稿为候选稿、派生历史样张作为Style Frame。
- 主Agent直接改CSS、改颜色、换模板后声称修复。
- 用“未冻结稿”“结构探索”“样张”“候选版”绕过生产门禁。
- 用自动验证通过、截图非空、页数正确替代审美复核。
- 运行Playwright/截图/导出/validation来证明修后效果；无subagent时QA必须停在流程阻断记录。

恢复生产前必须完成：

1. 《历史对话需求整合表》补齐本轮失败。
2. 《旧版否决门》列出不可继承项；被用户明确否决的视觉系统不得继续作为母版。
3. Skill/治理文档修订完成，并记录修订证据；若用户点名的是设计、动效、QA或参考库经验缺失，必须同步修改对应`references/*.md`，不得只在当前项目目录写失败审计。
4. 新的Style Frame或Page Spec流程获得真实subagent编制计划。
5. 用户明确同意恢复生产。

## 部门否决权

| 部门 | 可否决情形 | 必填字段 |
| :--- | :--- | :--- |
| 内容部门 | 页面设计压扁内容逻辑；为模板牺牲用户重点；讲授/演练/案例没有区分；复杂内容被做成泛化卡片 | 否决原因、被扭曲内容、建议重构方式 |
