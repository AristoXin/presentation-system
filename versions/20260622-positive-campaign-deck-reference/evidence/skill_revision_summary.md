# 20260622 正向多页 Campaign Deck 参考吸收证据

## 用户最新要求锁定

- 用户提供两张正向样例图片，并强调每张图片中都包含多页PPT内容。
- 用户要求学习其设计理念：UI设计不是简单对齐几个方块、放一堆配色、填入文字，而是 UI、动效等多部门协同的工作。

## 吸收结论

本次不保存原始微信缓存路径、截图文件或可识别来源，只沉淀脱敏设计经验。两张样例被归纳为“社媒营销 Campaign Deck 多页视觉系统”：

- 它们的价值不在于红蓝黄/黄黑等表层配色，而在于跨页视觉语法。
- 每张长图必须先拆成多页PPT角色，再提炼页面族谱。
- UI必须和语言内容、品牌审美、动效设计、交互工程、QA协同，才能形成可讲、可动、可交付的deck。
- 真实人物、商品、截图、口播Demo、箭头、步骤卡、标题牌、撕纸和网格都必须有信息角色，不能当随机装饰。
